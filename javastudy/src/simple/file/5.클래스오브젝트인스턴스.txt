1. class - 추상적인 설계도, 또는 구체적인 설계도  - 아메리카노와 카페라떼의 속성을 코드로 작성

추상 클래스 - 빵 - 사람이 상상해서 그 물건의 특성을 떠올릴 수 있지만 실제로 만들 수 없는것

		ex) 는 카레빵의 구체개념이지 빵 자체가 아니다

구체 클래스 - 카레빵 - 사람이 상상할 수 있고 제조법이 명확하다



2. Object - 실제로 존재할 수 있는 것을 상상 - 만들어질 카레빵을 머리 속으로 상상(구체 클래스)



3. Instance - 클래스를 통해 Object가 실체화 된 것 - 만들어진 카레빵 (new)
